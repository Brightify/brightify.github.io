Wealthico

Money, oil, stocks, gold, bitcoin. All in one place, simple and powerful.

When we were working on the Wealthico application, the greatest challenge was to create our own charts with fluid animations. A lot of effort was spent on tweaking these animations so that they feel right. Even though the first released version has only read access to the service, it still packs a lot of features like social login. A second version with even more will be released soon.
